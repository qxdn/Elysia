# Elysia
崩三bot，随缘开发ing。参考了[ATRI](https://github.com/Kyomotoi/ATRI)的一部分设计

# 功能
- [x] 复读
- [x] 随机色图
- [x] 查询图片出处
- [x] 生成色图(随着API跑路也不能用了，但还是可以用novelai的)
- [ ] 语音生成(在准备数据集)
## How to start

1. install requirements.
2. run your bot using `nb run` .
